# search

Trying to create a faster version of [ag](https://github.com/ggreer/the_silver_searcher) by taking advantage of goroutines for directory scanning and file processing

## Install
```bash
# create bin/search
make

# move to /usr/bin/local
make install
```

## Usage
```
ss [text or regex]
```
